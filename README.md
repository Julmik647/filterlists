# Filterlists for ad blockers
### Visual filterlists:
These filterlists need browser ad blocker like an extension or build-in one. They don't work with DNS ad blockers.
- [sweet.tv no login](https://github.com/Julmik647/sweet-tv-no-login/blob/main/sweet-tv-no-login-for-ad-blocker) - watch free live TV on [sweet.tv](https://sweet.tv/) without an account
- [SFW FMHY](https://github.com/Julmik647/filterlists/blob/main/sfw-fmhy) - Removes NSFW button from FMHY.net which has faster updates than FMHY.xyz
- [DocumentaryArea no watermark](https://github.com/Julmik647/filterlists/blob/main/documentaryarea-no-watermark) - remove watermark from DocumentraryArea
